# Horizon-Bow
Angular from Knockoutjs

This demo show the equivalent output in angular for a chart that was using knockoutjs.

First follow Prerequisites.md.

Then look at the original demo Html. 

Then using visual studio code follow the angular path of creation.

The demo is set over three branches that should be locked but clonable:
main - Only instructions and no code.
feature/chart-knockout-to-angular - Basic starting point.
feature/initial-setup - After completed instructions example code.
